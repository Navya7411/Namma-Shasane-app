Namma-Shasane 🏛️
AI-Powered Heritage Preservation & Inscription Translation App

An innovative Android application designed to identify, translate, preserve, and digitally map Karnataka’s historical stone inscriptions (“Shasanas”) using Artificial Intelligence and interactive heritage mapping.

📌 Overview

Namma-Shasane bridges the gap between ancient history and modern technology by transforming historical inscriptions into accessible digital knowledge.
The application allows users to discover nearby inscriptions, capture and upload stone images, receive AI-assisted translations, and contribute to cultural preservation through crowdsourced heritage mapping.

This project focuses on preserving Karnataka’s rich historical legacy while promoting awareness, tourism, and digital accessibility.

✨ Key Features
📷 Inscription Detection
Capture or upload photos of historical stone inscriptions.
🌐 AI-Based Translation   
Translate ancient inscriptions into modern Kannada/English.
🗺️ Interactive Heritage Mapping   
View nearby documented inscriptions using GPS integration.
🏺 Cultural Story View   
Learn historical context, dynasty details, and significance.
🚨 Preservation Reporting     
Report damaged or endangered heritage stones.
🔍 Search & Discovery  
Explore inscriptions by location, dynasty, or keywords.

📱 Application Workflow
User captures an inscription image.
OCR extracts inscription text.
AI module processes and translates content.
Historical details are displayed in story format.
Location is added to the heritage map.
Users can report preservation issues.
